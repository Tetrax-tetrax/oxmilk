Welcome to the official repository for **Oxmilk's Portfolio** website!

## Overview

This site is meant to be the hub for Oxmilk's portfolio items, such as commission info, game dev blogging, web comics, and more.
It features a highly customized, "Old Web" aesthetic with interactive elements, marquee animations, and a dedicated guestbook.

## Project Structure

```
assets/             # Graphics, blinkies, and other UI elements
css/                # Style sheets for marquee, bounce, and other elements
font-family/        # Custom bubble club font
scripts/            # Compiled typescript for banner sizing and bounce physics
index.html          # Main entry point
```
